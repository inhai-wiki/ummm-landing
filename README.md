# Ummm Landing Page

Ummm 是一款 macOS 语音转文字工具，让你通过说话代替打字，效率提升 3 倍。

## 功能特点

- **按住即录** - 按住 fn 键说话，松开即输入，无需点击开始/停止按钮
- **光标定位** - 文字自动输入到光标所在位置，适用于任何应用
- **实时转写** - 边说边转，即时看到结果
- **隐私优先** - 默认使用本地识别，数据不离开你的 Mac
- **云端可选** - 可配置 API Key 使用更精准的云端识别
- **中文优化** - 深度优化中文语音识别效果

[阿里云百炼 - 语音模型体验中心](https://bailian.console.aliyun.com/cn-beijing/?source_channel=%22ummm%22/?tab=model#/efm/model_experience_center/voice)

## 技术栈

- HTML5
- CSS3
- Vanilla JavaScript
- 支持中英文双语切换

## 本地预览

直接用浏览器打开 `index.html` 即可预览，或使用本地服务器：

```bash
# 使用 Python
python -m http.server 8080

# 或使用 Node.js
npx serve .
```
